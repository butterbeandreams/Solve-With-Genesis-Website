## How to Deploy (Easiest Method)

### Option 1: Netlify Drop (Recommended — 60 seconds)
1. Go to: https://app.netlify.com/drop
2. Drag and drop this entire `website` folder
3. You instantly get a live URL (e.g. `something.netlify.app`)
4. Optional: Connect your own domain later

No account required for the first deploy. Completely free.

### Option 2: Vercel
1. Go to vercel.com
2. Drag and drop the folder or connect GitHub

### Option 3: Cloudflare Pages
1. Go to pages.cloudflare.com
2. Upload the folder
